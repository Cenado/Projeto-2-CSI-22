## Instruções

O projeto foi concebido e testado nas versões `3.12` e `3.13` do Python. 

Execute `git clone` em uma pasta de preferência com o repositório remoto do projeto. Em seguida, execute na pasta do projeto `pip install -r requirements.txt` para instalar as dependências do projeto. É recomendado a criação de um ambiente virtual.

Por fim, execute `python main.py` para rodar a aplicação. Bom jogo!
